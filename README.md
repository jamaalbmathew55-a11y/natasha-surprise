# Natasha Surprise 💕

A sweet interactive surprise webpage with an animated heart and floating bouquets!

## Features
- 💖 Animated beating heart with glow effect
- 🌸 Floating flower bouquets that react to the heart
- 📱 Mobile responsive design
- ✨ Smooth animations and transitions

## How to Use
1. Click the "Click Me" button to reveal the heart
2. Use "Add Bouquet" to add more flowers
3. Use "Remove Bouquet" to remove flowers

## To Add Your Own Images
Create an `assets/` folder and add your flower images:
- `nr-one.jfif`
- `nr-two.jfif`
- `nr-three.jfif`
- `nr-four.jfif`
- `nr-five.jfif`

Then push them to the repository.
